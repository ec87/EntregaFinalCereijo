## Que es mi simulador


El presente contiene una estructura simple de HTML, con un documento js enlazado mediante un enlace.

La temática que se decidió abordar es un sitio web de control del periodo menstrual. Será posible realizar el cálculo de las próximas 12 menstruaciones cuando la persona menstruante ingrese los siguientes datos:

* Fecha de la última menstruación (formato Date).
* Duración del periodo menstrual.

Se piden ademas más datos que futuramente optimizaran el procesamiento de datos, y ampliaran la funcionalidad del sitio web (poder marcar en un calendario los dias de sangrado, ovulación, etc).

Ademas, cuenta con puntos de control de ingresos de datos invalidos.

## Que funcionalidad incluye para el usuario

- Calculo de las siguientes 12 menstruaciones.
- Filtrado de lista de menstruaciones de un mes solicitado.

## Por donde se le pide información al usuario

Las interacciones se realizan por medio de uso de prompts.

## Como se le devuelve la info al usuario

Las informaciones se devuelven por medio de console.log.
